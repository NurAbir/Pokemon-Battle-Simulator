###  Overview  
This project is a **Pokémon Battle Simulator** where players can choose Pokémon, set moves, and battle in real-time or turn-based mode. It’s designed to replicate the strategy and excitement of Pokémon Showdown while keeping the codebase simple and modular for learning or extension.  

---

###  Features  
-  **Team Builder** – Select Pokémon, customize moves, abilities, and items  
-  **Battle Engine** – Calculates damage, status effects, and turn order  
-  **Real-Time or Turn-Based** play options  
-  **Clean UI** built for intuitive gameplay  
-  **Battle Logs** and stat summaries  
-  **Multiplayer Support (optional)** using sockets or REST API  


---

###  Tech Stack  
- **Frontend:** HTML, CSS, JavaScript (or React if used)  
- **Backend:** Node.js / Express (if multiplayer)  
- **Database:** MongoDB / JSON storage for teams and stats  
- **Version Control:** Git & GitHub
